# DJ Learning Lab

An interactive web app that actually teaches DJing, not just a checklist.

## What it includes

- 10-module structured curriculum across foundations, beatmatching, phrasing, EQ, transitions, harmonic mixing, performance, workflow, and growth
- Interactive lesson player for every lesson:
  - Concept explanation
  - Step-by-step practice drill
  - Multiple-choice knowledge check with instant feedback
  - Completion unlocked only after practice + correct answer
- Progress tracking persisted in localStorage
- Practice console with 5-minute timer + random drills
- Song Match Agent that recommends mix candidates by BPM, Camelot key compatibility, and energy level
- Internet song lookup in Song Match Agent (fetches live tracks + preview links from public API)
- Session notes saved locally

## Run

Open `index.html` in your browser.
